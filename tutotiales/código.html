código

Ejemplo de código de programación:

x = 5; y = 6; z = x + y;

El elemento de código no conserva los espacios en blanco ni los saltos de línea.

Para solucionar esto, puedes colocar el elemento de código dentro de un elemento pre:


x = 5;
y = 6;
z = x + y;
