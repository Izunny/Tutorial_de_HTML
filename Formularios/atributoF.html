formulario de entrada

El atributo de formulario especifica el formulario al que pertenece un elemento de entrada.



El campo "Apellido" que aparece a continuación está fuera del elemento de formulario, pero sigue siendo parte del formulario.

entrada formenctype

El atributo formenctype especifica cómo deben codificarse los datos del formulario cuando se envían.



formmethod

El atributo formmethod define el método HTTP para enviar datos del formulario a la URL de la acción.
