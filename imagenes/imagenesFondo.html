imagen de fondo

Puede especificar imágenes de fondo
para cualquier elemento HTML visible.
En este ejemplo, la imagen de fondo
se especifica para un elemento p.
De forma predeterminada, la imagen de fondo
se repetirá en la dirección
en la que sea más pequeña que el elemento
en el que se especifica. Intente cambiar el tamaño de la
ventana del navegador para ver cómo
se comporta la imagen de fondo.
