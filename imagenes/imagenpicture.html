picture

Cambie el tamaño del navegador para ver distintas versiones de la imagen que se cargan en distintos tamaños de ventana gráfica. El navegador busca el primer elemento de origen en el que la consulta de medios coincide con el ancho de la ventana gráfica actual del usuario y obtiene la imagen especificada en el atributo srcset

El elemento img es necesario como última etiqueta secundaria del bloque de declaración de imagen. El elemento img se utiliza para proporcionar compatibilidad con versiones anteriores para navegadores que no admiten el elemento de imagen o si ninguna de las etiquetas de origen coincide.
